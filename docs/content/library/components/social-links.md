# Social Links

**SocialLinks -** The SocialLinks component is a reusable UI element for the FastLane website, designed to display a row of social media icons linking to external profiles or actions. Built using Radix UI and Tailwind CSS, it enhances user engagement by providing quick access to social platforms. The component is configurable via Sitecore for link management and adapts to various screen sizes.

**Description:** A horizontal layout containing icons for social media platforms (e.g., email, Facebook, Instagram, LinkedIn, X, YouTube).

**Functionality:** Each icon is clickable, redirecting users to the corresponding social media page or triggering an action (e.g., opening an email client). Links are managed through Sitecore, ensuring flexibility and easy updates.

## Field Details

**Template Name:** SocialLinks
**Item Path:** `/sitecore/templates/Feature/FastLane/Renderings/SocialLinks/SocialLinks`

| Field Name | Sitecore Field Type | Description |
|------------|-------------------|-------------|
| SocialIcon | Image | Icon / Image for the Social Media |
| SocialLink | General Link | General Link for the Social URL. |
