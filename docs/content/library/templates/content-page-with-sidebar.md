# Content Page with Sidebar

**Content Page with Sidebar:** The Content Page with Sidebar template, inherited from the base "Page" template in Sitecore, is designed for in-depth content delivery with an accompanying sidebar.

This template features a main content area for detailed text, images, or media, paired with a sidebar that can include navigation links, related content, or quick access items.

*   **Structure**: Includes a header, main content area for rich text and media, and a sidebar for contextual links.
    
*   **Use Case**: Perfect for detailed articles or service descriptions with supporting navigation.
