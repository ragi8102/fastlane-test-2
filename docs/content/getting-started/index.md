### Common Prerequisites (All Roles)

- **Node.js 18.x or higher** - [Download](https://nodejs.org/)
  - Verify installation: `node --version`
  - npm is included with Node.js
- **Git** - [Download](https://git-scm.com/)
  - For version control and repository access
- **Sitecore XM Cloud Environment Access**
  - Active XM Cloud subscription
  - Organization and tenant credentials
  - Access to XM Cloud Deploy portal
- **Modern web browser** (Chrome, Firefox, Edge, or Safari)
  - For accessing tools and previewing content
