## ðŸ”„ Migration from JSS

### Key Changes in Content SDK

**Hook Changes:**
```typescript
// JSS (Old)
import { useSitecoreContext } from '@sitecore-jss/sitecore-jss-nextjs';
const { sitecoreContext } = useSitecoreContext();
const isEditing = sitecoreContext.pageEditing;

// Content SDK (New)
import { useSitecore } from '@sitecore-content-sdk/nextjs';
const { page } = useSitecore();
const isEditing = page.mode.isEditing;
```

**Component Registration:**
```typescript
// JSS (Old)
import { componentFactory } from '@sitecore-jss/sitecore-jss-nextjs';
const componentFactory = {
  Button: ButtonComponent,
  Hero: HeroComponent,
};

// Content SDK (New - AUTO-GENERATED)
// .sitecore/component-map.ts (generated by sitecore-tools:generate-map)
export const componentMap = new Map<string, NextjsContentSdkComponent>([
  ['Button', Button],
  ['Hero', Hero],
  // ... all components auto-registered
]);
```

**Page Mode Detection:**
```typescript
// Enhanced granular mode detection
const { page } = useSitecore();

if (page.mode.isNormal) { /* Normal website view */ }
if (page.mode.isPreview) { /* Preview mode */ }
if (page.mode.isEditing) { /* Content editing */ }
if (page.mode.isDesignLibrary) { /* Design Library showcase */ }
if (page.mode.isVariantGeneration) { /* AI variant generation */ }
```
